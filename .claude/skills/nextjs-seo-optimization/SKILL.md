# Next.js SEO Optimization Skill

Esta skill es una referencia exhaustiva para implementar SEO técnico avanzado en aplicaciones Next.js utilizando el App Router y TypeScript. Cubre desde la API de Metadata hasta la optimización de rendimiento y datos estructurados.

## Contenido

1.  **[Metadata API & Open Graph](./01-metadata-opengraph.md)**
    *   Configuración de `metadata` estática.
    *   Uso de `generateMetadata` para páginas dinámicas.
    *   Open Graph (OG) Images y Twitter Cards.
    *   Canonical URLs y templates de títulos.

2.  **[Structured Data (JSON-LD)](./02-structured-data.md)**
    *   Implementación de Schemas (Article, Product, Breadcrumb).
    *   Validación con Google Rich Results.
    *   Tipos seguros para JSON-LD.

3.  **[Sitemaps & Robots](./03-sitemap-robots.md)**
    *   Generación dinámica de `sitemap.ts`.
    *   Configuración de `robots.ts`.
    *   Gestión de crawling e indexación.

4.  **[Performance & Images](./04-performance-images.md)**
    *   Optimización con `next/image` para LCP.
    *   Responsive images y formatos modernos (AVIF/WebP).
    *   Core Web Vitals y su impacto en el ranking.

5.  **[Internationalization (i18n) SEO](./05-i18n-seo.md)**
    *   Etiquetas `hreflang`.
    *   Metadatos específicos por idioma.
    *   Estructura de URLs internacional.
