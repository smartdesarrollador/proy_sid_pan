Generate a comprehensive project report:

1. Analyze the current state of the codebase
2. Check for pending TODOs and FIXMEs
3. Review test coverage status
4. List recent changes (git log)
5. Identify potential tech debt
6. Save the report to `reports/` with today's date

Format: Markdown with sections and actionable items.
