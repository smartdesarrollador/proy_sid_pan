# Project Glossary

| Term | Definition |
|------|-----------|
| **Agent** | An AI entity that can use tools and make decisions autonomously |
| **RAG** | Retrieval-Augmented Generation - combining search with LLM generation |
| **Embedding** | A vector representation of text for similarity search |
| **Prompt Template** | A reusable prompt pattern with variable placeholders |
| **Chain** | A sequence of LLM calls where each step builds on the previous |
| **Tool** | A function that an agent can invoke to interact with external systems |
| **Token** | The basic unit of text processing for LLMs (~4 chars in English) |
| **Context Window** | The maximum number of tokens an LLM can process in one request |
| **Fine-tuning** | Training an existing model on domain-specific data |
| **Vector DB** | A database optimized for storing and querying embeddings |
